Dualword-PMC - [PubMed Central](http://www.ncbi.nlm.nih.gov/pmc/) browser written in [C++](http://isocpp.org).

Features:
 - web browser with access to PubMed Central website
 - PDF files are automatically stored in SQLite database
 - offline PDF viewer
 - full-text search
 - image extraction
